## Tarea: Aplicación Web con Vue y Vuetify  

### Descripción

Desarrollar una aplicación web de presentación personal o portafolio utilizando **Vue 3** (framework progresivo de JavaScript para construir interfaces de usuario basadas en componentes) y **Vuetify 3** (framework de componentes UI basado en Material Design para Vue).

La aplicación deberá:

- Consumir imágenes aleatorias desde la API de **Picsum** (servicio web que proporciona imágenes de prueba aleatorias).
- Permitir actualizar dinámicamente las imágenes mediante un botón.
- Mostrar los metadatos de cada fotografía (información descriptiva asociada a la imagen, como autor, dimensiones o identificador).

El objetivo es aplicar conceptos de:
- Componentización
- Consumo de APIs REST
- Reactividad
- Manejo de eventos
- Diseño responsivo
